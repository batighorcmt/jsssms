Place the following real asset files here:
- icon.png          (App launcher icon 1024x1024 or square high-res)
- Splash.gif        (Animated splash; keep dimensions <= 512x512 for performance)
- loading.gif       (Small looping loader e.g. 64x64)

After copying files run:
flutter pub get
flutter pub run flutter_launcher_icons
flutter pub run flutter_native_splash:create
